A recommender that looks at how an album sounds and how it feels, not which genre tag it carries. The hypothesis: my favorite albums share sonic and mood properties that genre labels paper over, and a system listening for those properties directly should surface cross-genre matches a tag-based recommender never would.

## Approach

Each album sits in a feature space built from audio characteristics and mood descriptors. A KNN lookup returns the nearest albums to whatever you give it as a seed. Next.js and Chakra UI for the frontend, a Flask service for the recommender, deployed on Vercel and Heroku.

## What's interesting

The cross-genre hits. A jazz record pairing with an ambient electronic record because both move at the same pulse with the same dynamic range. That connection isn't in any tag.
