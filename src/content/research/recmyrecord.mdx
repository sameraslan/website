A small recommender built as a probe for a question: can audio features and mood descriptors predict the cross-genre listens I actually like, in a way genre and artist tags can't?

## What this is testing

Genre and artist tags collapse useful information. Two records that move at the same pulse and sit in the same emotional register can land in different tag clusters and stay invisible to each other. Audio and mood should surface those pairings directly.

## Setup

Each album is represented by audio features and a small set of mood descriptors. A KNN over that space returns nearest neighbors for a seed album. The interface lets you compare neighbors from audio-only vs. mood-only vs. combined retrieval, which is where the cross-genre pairings show up clearest.

## Where this points

This is a small, opinionated probe, not a paper. It informed how I think about the [music map](/music) on this site, which uses similar features at a different scale.

Live at [recmyrecord.com](https://www.recmyrecord.com).
